# Foobow User Stories

## New User

- As a new user, I want to understand the app purpose quickly so I know it is for small symbolic good deeds and emotional comfort.
- As a new user, I want to choose language, region, and privacy preference so the experience feels safe and personal.
- As a new user, I want to complete a first deed in under one minute so I can feel the value immediately.
- As a new user, I want donation and karma language to be clear so I understand what is symbolic and what is real-world impact.

## Daily User

- As a daily user, I want to check in with my mood so the app can recommend a meaningful deed.
- As a daily user, I want a short ritual that does not feel like a chore.
- As a daily user, I want gentle streaks and badges so I feel progress without pressure.
- As a daily user, I want to write a private reflection after a deed so I can process my feelings.

## Map Explorer

- As an explorer, I want to browse good-deed spots around the world so the app feels like meaningful travel.
- As an explorer, I want map layers by theme so I can focus on animals, environment, elders, donations, or emotional support.
- As an explorer, I want to collect map stamps so returning to the map feels rewarding.
- As an explorer, I want famous locations and cultural moments to have special rituals.

## Virtual Deed Participant

- As a participant, I want to release a virtual fish into a lake or river so I can experience symbolic 放生 without harming ecosystems.
- As a participant, I want to help an elderly person cross a virtual street so I can complete a familiar good-will action online.
- As a participant, I want to send anonymous blessings so I can comfort others without social pressure.
- As a participant, I want each action to be simple, respectful, and visually satisfying.

## Social User

- As a social user, I want to join kindness chains so I can feel connected to others.
- As a social user, I want low-pressure reactions such as bless, support, thank you, and same feeling.
- As a social user, I want family, friend, school, or company groups for shared challenges.
- As a social user, I want reporting and blocking tools so the community stays safe.

## Private User

- As a private user, I want anonymous mode so my actions are not attached to my identity.
- As a private user, I want quiet mode so I can avoid rankings.
- As a private user, I want my journal to be private by default.
- As a private user, I want to export or delete my data.

## Donor

- As a donor, I want to see verified causes so I can trust where money goes.
- As a donor, I want small donation options so giving feels accessible.
- As a donor, I want receipts and impact updates.
- As a donor, I want clear wording that donations support real causes but do not purchase karma.

## Moderator / Admin

- As a moderator, I want to review reported blessings, comments, campaigns, and profiles.
- As a moderator, I want to flag fake charities or misleading donation claims.
- As a moderator, I want to manage sensitive cultural, memorial, disaster, and youth content.
- As an admin, I want category and campaign analytics without exposing private user journals.
